# Diesel OS Lab — Branding Guidelines

## Objetivo

Este documento define a identidade visual oficial do projeto **Diesel OS Lab**, para manter consistência entre logo, avatar, splash, wallpaper, interface e materiais futuros.

---

## Conceito da marca

O Diesel OS Lab deve transmitir:

- tecnologia premium
- performance
- sofisticação
- engenharia
- identidade autoral
- estética dark luxury
- gaming presente de forma refinada

A marca **não** deve parecer:

- genérica
- caricatural
- neon exagerado
- poluída
- e-sports comum
- agressiva demais visualmente

---

## Elemento central

O elemento principal da identidade é:

**Monograma da letra D em metalizado azul refinado**

Características desejadas:

- geometria elegante
- linhas precisas
- aparência sólida
- equilíbrio visual forte
- leitura fácil
- presença premium
- sensação tecnológica e atemporal

---

## Referência visual principal

A referência visual principal aprovada para o projeto é a composição com:

- monograma D metálico azul ao centro
- mouse à esquerda
- controle à direita
- mira sutil
- texto "DIESEL OS LAB"
- subtítulo "TECHNOLOGY & GAMING PLATFORM"

Essa composição serve como base para derivações futuras de:

- logo
- avatar
- splash
- wallpaper
- branding do sistema
- materiais de apresentação

---

## Paleta visual

### Fundo

- preto carvão
- grafite profundo
- antracite escuro

### Destaques

- azul metálico frio
- azul aço
- azul refinado de aparência tecnológica

### Apoio

- prata fria
- aço escovado azulado
- cinza metálico escuro

---

## Materiais e acabamento

A linguagem visual deve lembrar:

- metal escovado
- titânio azulado
- superfícies técnicas premium
- brilho controlado
- profundidade sutil
- contraste limpo

Evitar:

- glow excessivo
- reflexos exagerados
- plástico brilhante
- aparência infantil
- visual excessivamente chamativo

---

## Gaming na identidade

Gaming faz parte do projeto, mas deve aparecer com controle.

Permitido:

- mouse
- controle
- mira sutil
- sinais discretos de performance
- atmosfera tecnológica gamer premium

Evitar:

- excesso de elementos gamer
- visual arcade
- exagero neon
- linguagem visual caricatural
- poluição de cena

Regra principal:

**gaming deve complementar a marca, nunca dominar o monograma D**

---

## Tipografia

A tipografia deve parecer:

- moderna
- tecnológica
- limpa
- premium
- estável
- madura

Preferências:

- sans-serif forte e limpa
- espaçamento equilibrado
- boa legibilidade
- aparência institucional premium

## Padrão de nomes

O padrão oficial de nomes é:

`diesel-os-lab-[categoria]-[variante]-v[número].[extensão]`

Exemplos:

- `diesel-os-lab-logo-full-v2.png`
- `diesel-os-lab-avatar-github-v2.png`
- `diesel-os-lab-splash-dark-v2.png`
- `diesel-os-lab-wallpaper-dark-4k-v3.jpg`

---

## Regras de composição

- o monograma D é sempre o foco principal
- os elementos laterais devem apoiar, não competir
- a composição deve parecer premium
- o fundo deve reforçar profundidade e contraste
- a marca deve manter leitura forte em tamanhos diferentes

---

## Regra de coerência

Todo asset novo do Diesel OS Lab deve parecer derivado da referência principal aprovada.

Se uma peça visual parecer de outro projeto, ela está errada.

---

## Princípio final

**Diesel OS Lab deve parecer uma plataforma premium de tecnologia e gaming, não apenas um tema bonito.**
